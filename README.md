# 🤖 CyberFarm: TON-ферма с PvP-рейдами

> Telegram Mini App игра на React + TON — тапай, управляй роботами и кради ресурсы у друзей

## 📖 О проекте

**CyberFarm** — это гибрид Clicker'а и тактической стратегии в Telegram. В отличие от классических тапалок, здесь вы не просто кликаете по экрану, а управляете роботами-фермерами, защищаете свой урожай и можете красть ресурсы у других игроков.

| Механика | Описание |
| :--- | :--- |
| 🖱️ **Tap-to-Earn** | Кликайте по дронам, чтобы ускорять добычу и чинить сломанных роботов |
| 🤖 **Пассивный доход** | Роботы добывают токены даже в офлайн-режиме |
| ⚔️ **PvP-рейды** | Крадите ресурсы у друзей или теряйте своих дронов при неудачной атаке |
| 🛡️ **Защита** | Стройте турели и щиты за накопленные токены |
| 🏆 **Лиги** | От бронзового фермера до золотого магната — выше лига → больше доход |

## 🎮 Игровой процесс (пошагово)

1. **Старт** — вы получаете базового дрона и 3 единицы энергии.
2. **Фарм** — дрон добывает монеты автоматически. Клики ускоряют процесс x2.
3. **Апгрейд** — тратите монеты на улучшение дронов, турелей и щитов.
4. **Рейд** — атакуете ферму друга. Если его защита слабее — крадёте 10-20% его текущего баланса.
5. **Защита** — отбиваете атаки других игроков и смотрите лог сражений.
6. **Вывод** — подключите TON-кошелёк и выводите заработанные токены.

## 💰 Экономика и заработок

| Способ заработка | Механика |
| :--- | :--- |
| **Tap-to-Earn** | Клик = +0.1 монеты (ускоряется апгрейдами) |
| **Пассивный доход** | Дроны капают от 1 до 100 монет/час в зависимости от уровня |
| **PvP-ставки** | Перед рейдом можно заключить пари — победитель забирает банк |
| **Лиги** | Чем выше лига, тем больше монет за каждый клик и пассивный доход |
| **Рефералы** | +5% от дохода каждого приглашённого друга |

## 🚀 Быстрый старт

```bash
# Клонируйте шаблон (замените на ваш репозиторий)
git clone https://github.com/your-username/cyberfarm.git
cd cyberfarm

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev
```

## 🔧 Настройка бота в Telegram

1. Напишите [@BotFather](https://t.me/BotFather) → создайте нового бота командой `/newbot`
2. Получите API-токен
3. В BotFather отправьте команду `/setdomain` и укажите URL вашего приложения (например, `https://cyberfarm.vercel.app`)
4. Нажмите `/setuserpic` и загрузите иконку для бота
5. Готово — открывайте бота и играйте
